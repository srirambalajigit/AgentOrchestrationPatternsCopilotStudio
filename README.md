# AgentOrchestrationPatternsCopilotStudio
This repo contains sample files for the video series: Agent Orchestration Patterns using Copilot Studio https://www.youtube.com/playlist?list=PL3yFlHSZicl87fdSSZ4kuuUVNV2ypOxhv
This repo walks through the various agent orchestration patterns and how you can implement each pattern in Copilot Studio.  It covers orchestration patterns like Sequential, Concurrent, Group chat, Handoff and Magentic patterns.
